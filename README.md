# 🧠 AI Interview App (Mem0 + LangChain + Groq)

An AI-powered interview preparation assistant that learns from users over time using a persistent memory layer.  
It personalizes questions, avoids repetition, and focuses on weak areas to improve interview readiness.

---

## 🚀 Features

- 🧠 **Persistent Memory (Mem0)** – Remembers user interactions across sessions  
- 🤖 **LLM-Powered Coaching** – Uses Groq (LLaMA 3) for fast responses  
- 📚 **Context-Aware Questions** – Tailors questions based on past conversations  
- 🔍 **Semantic Search** – Uses Qdrant vector DB for memory retrieval  
- 🌐 **Streamlit UI** – Simple and interactive chat interface  

---

## 🛠️ Tech Stack

- **LangChain** – Orchestration  
- **Groq (LLaMA 3)** – LLM  
- **Mem0** – Memory layer  
- **Gemini Embeddings** – Semantic understanding  
- **Qdrant** – Vector database  
- **Streamlit** – Frontend
